# Installation & Setup

Getting started with **OptionBay** is simple and follows the standard WordPress plugin installation process. This guide will walk you through installing the plugin and ensuring your WooCommerce store is ready for custom product options.

## Prerequisites

Before installing OptionBay, please ensure your site meets the following requirements:

| Requirement | Minimum Version |
| :--- | :--- |
| **WordPress** | 5.6 or higher |
| **WooCommerce** | 6.1 or higher |
| **PHP** | 7.0 or higher |

> [!IMPORTANT]
> OptionBay requires WooCommerce to be installed and activated. The plugin will not function without an active WooCommerce installation.

## Installation Steps

### 1. Upload and Activate
1. Log in to your WordPress Admin Dashboard.
2. Navigate to **Plugins > Add New**.
3. Click the **Upload Plugin** button at the top of the page.
4. Choose the `optionbay.zip` file from your computer and click **Install Now**.
5. Once the installation is complete, click **Activate Plugin**.

### 2. Verify Menu Access
After activation, you should see the following new entries in your WordPress admin:

| Location | Purpose |
| :--- | :--- |
| **Products > Options** | Create and manage your Option Groups — this is where you build your forms. |
| **WooCommerce > Settings > OptionBay** | Global display and performance settings for the plugin. |

> [!TIP]
> If you don't see the "Options" menu under Products, try refreshing the page or clearing your browser cache. WordPress sometimes needs a hard refresh after plugin activation.

## Initial Configuration

Once the plugin is activated, it's recommended to check the global settings to ensure the display matches your theme:

1. Navigate to **WooCommerce > Settings** and click on the **OptionBay** tab.
2. Configure your **Options Orientation** (Vertical or Horizontal) to match your product page layout.
3. Set your preferred **Font Sizes** for labels and inputs to align with your theme's typography.
4. If you plan to use swatches, configure the **Swatch Size** and **Swatch Radius** settings.
5. Save your changes using the native WooCommerce **Save changes** button at the bottom.

## What's Next?

2. **[Build Your Fields](./builder)** — Use the drag-and-drop Addon Builder to add fields.
