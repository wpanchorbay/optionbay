# Screenshot Requirements for TubeBay Documentation

Please capture the following screenshots and place them in the `docs/public/` folder of the VitePress site exactly matching these filenames:

1. `logo-2.png`
   - Description: The official TubeBay logo (already requested but double checking).

2. `tubebay-dashboard-mockup.png`
   - Description: A beautiful, high-quality showcase image of the TubeBay plugin dashboard. This is the hero image for the home page.

3. `setup-wizard-step1.png`
   - Description: A screenshot of the first step of the Onboarding Setup Wizard where the user enters their YouTube API Key and Channel ID, ideally showing a successful "Connection Tested!" green message.

4. `setup-wizard-step2.png`
   - Description: A screenshot of the second step of the Onboarding Setup Wizard showing the global default configuration options (Placement, Autoplay, player controls toggles).

5. `channel-library-overview.png`
   - Description: A full screenshot of the "Channel Library" page in Grid View, showing the search bar, sort dropdown, and a gallery of video thumbnails.

6. `video-card-actions.png`
   - Description: A cropped screenshot focusing on a single video card inside the Channel Library showing the action buttons on hover (the eye icon and clipboard icon).

7. `product-metabox.png`
   - Description: A screenshot of the WooCommerce product edit screen, specifically focused on the right sidebar "TubeBay Video" metabox. Ideally show it *after* a video has been assigned to the product, displaying the small video preview, UI buttons, and the Muted Autoplay toggle.

8. `storefront-video-display.png`
   - Description: A screenshot of the live WooCommerce storefront product page showing a TubeBay video displaying inside the product gallery (or replacing the main image) with the Video Façade play button visible.

9. `copy-shortcode.png`
   - Description: A screenshot showing the action of hovering over a video card in the Channel Library and clicking the "Copy Shortcode" clipboard icon. The tooltip saying "Copy Shortcode" should be visible.

10. `settings-dashboard.png`
    - Description: A wide screenshot of the full "Settings" page in the TubeBay admin dashboard, showing the layout of the Connection Card, Sync Card, Placement Card, etc.

11. `placement-settings.png`
    - Description: A cropped screenshot focusing squarely on the "Placement Settings" card, showcasing the visual selector layout for choosing where WooCommerce will inject the video.
